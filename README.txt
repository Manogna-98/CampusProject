# CampusProject

## Overview

CampusProject is a web application designed to connect students, faculty, and alumni of a college or university. It provides a platform for collaboration, resource sharing, and communication within the campus community.

## Features

- **Home Page**: Introduction to the platform and highlights of campus life, facilities, and events.
- **About Us Page**: Information about the mission, team, and contact details.
- **Contact Us Page**: A form to get in touch with the campus administration.
- **Sign Up Page**: A form for new users to create an account.
- **Login Page**: A form for existing users to log in to their accounts.

## File Structure

Pages

Home Page (index.html)
Description: The main landing page of the website.
Features: Carousel showcasing campus highlights, navigation bar, and sections for campus news and events.

About Us Page (about.html)
Description: Provides information about the mission, team, and contact details.
Features: Consistent styling with the home page, navigation bar, and main content sections.

Contact Us Page (contact.html)
Description: A form for users to get in touch with the campus administration.
Features: Form fields for name, email, and message. Displays a success modal upon form submission.

Sign Up Page (signup.html)
Description: A form for new users to create an account.
Features: Form fields for first name, last name, email, date of birth, password, and mobile number. Displays a success modal upon form submission.

Login Page (login.html)
Description: A form for existing users to log in to their accounts.
Features: Form fields for username/email and password. Displays a success modal upon form submission.

Stylesheets
Main Stylesheet (style.css)
Description: Contains the main styles for the home page.
Features: Styles for the body, container, header, navigation bar, main content, and footer.

About Us Stylesheet (about.css)
Description: Contains styles specific to the About Us page.
Features: Consistent styling with the home page, styles for the main content sections.

Sign Up Stylesheet (signup.css)
Description: Contains styles specific to the Sign Up and Login pages.
Features: Responsive form styles, input fields, and buttons.
